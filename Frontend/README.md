# Ticketnator Frontend 

### Overview
Ticketnator Frontend is a tool for managing tickets, connecting seamlessly to the backend.

### Getting Started
1. Clone the repo.
2. Run `npm install` and `npm run dev`.
3. Visit `http://localhost:3000`.

### Troubleshooting
Ensure the backend is running!
